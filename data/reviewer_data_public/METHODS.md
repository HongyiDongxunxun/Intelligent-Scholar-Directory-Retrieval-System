# Verification Scope

| Evidence | Can verify | Cannot verify |
|---|---|---|
| E1 normalized features | Five ranking methods, Full formula, Top-10 pooling, entity counts | Original retrieval recall or real entity identity |
| E2 sanitized repeats | Stability of confirmed execution and variation after parsing | Population-level model accuracy |
| E3 aggregate snapshot | Reported aggregate values and interpretation boundaries | Missing historical request-level observations |
| E3 synthetic measurements | Statistical implementation | Historical performance |
| Synthetic fixture | Data contracts, aggregation, filters, edge handling | Distribution of the confidential database |

Exact feature combinations can remain structurally distinctive even after direct identifiers are removed. `validation_report.json` quantifies this linkage-risk indicator. The public package intentionally excludes the relationship-level reconstruction data reserved for controlled review.
